A basic pixel painter built with React, TypeScript, Vite and uses 'html2canvas' to render and download the data as a .PNG file.

## About the Project

This simple project lets a user choose a limited sized grid(either x16, x32, x64) and draw on a canvas using CSS Grid.
Currently you can view it live here: https://pixel-painter-hdvt.onrender.com/

## Installation

Simply clone the repo and run:'

`cd pixel-painter`  
`npm install`  
`npm run dev`
